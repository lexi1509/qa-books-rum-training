## Найденные баги

### Баг 1: Ввод минимального размера фамилии
- **Шаги:** Ввести значение «По»
- **Ожидаемый результат:** Принято
- **Фактический результат:** Отклонено
- **Причина:** Проверить границы входных значений (2 символа)

### Баг 2: Ввод фамилии + цифры
- **Шаги:** Ввести значение «Иванов123»
- **Ожидаемый результат:** Отклонено
- **Фактический результат:** Принято
- **Причина:** Программа обрабатывает только фамилию, цифровые обозначения не учитываются в валидации

### Баг 3: Ввод значения на арабской раскладке
- **Шаги:** Ввести значение `أَلِف`
- **Ожидаемый результат:** Отклонено
- **Фактический результат:** Принято
- **Причина:** Неверная обработка различных раскладок
