# Схема состояний и переходов: заявка на кредит

## Условия
В момент подачи пользователем заявления на кредит создается заявка в статусе **«Создана»**.

### Переходы

1. **После автоматической проверки кредитной истории заёмщика:**
   - Если кредитная история **«плохая»** → статус **«Отклонена»**
   - Если кредитная история **«средняя»** → статус **«На уточнении»**
   - Если кредитная история **«хорошая»** → статус **«Одобрена»**

2. **Из статуса «На уточнении»:**
   - Если ежемесячный доход клиента **выше 10 000** → статус **«Одобрена»**
   - Если ежемесячный доход клиента **ниже 10 000** → статус **«Отклонена»**

3. **Из статуса «Одобрена»:**
   - Если клиент **отказывается** получить кредит → статус **«Отклонена»**
   - Если клиент **получает** одобренный кредит → статус **«Выдана»**
