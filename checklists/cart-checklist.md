# Чек-лист проверки корзины интернет-магазина «Книги Рум»

## Общая информация
- **Тестируемый сайт:** [Книги Рум](http://188.120.241.222)
- **Цель:** Составление чек-листа для проверки базовой функциональности корзины

---

## Чек-лист

| № | Проверка |
|---|----------|
| 1 | Добавление товара в корзину со страницы каталога |
| 2 | Добавление товара в корзину с детальной страницы товара |
| 3 | Счетчик корзины в хедере изменяется при добавлении нескольких разных товаров |
| 4 | Счетчик корзины в хедере не изменяется при добавлении нескольких одинаковых товаров |
| 5 | Интерфейс страницы «Корзина» (присутствуют все требуемые поля) |
| 6 | Корзина по умолчанию пуста при новом входе пользователя |
| 7 | Применение валидного промокода |
| 8 | Применение невалидного промокода |
| 9 | Расчет итоговой суммы при применении валидного промокода |
| 10 | Применение карты лояльности к заказу |
